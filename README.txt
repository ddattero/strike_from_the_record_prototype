Strike From the Record - Prototype

What this is:
A tiny playable vertical slice for the courtroom deckbuilder idea.
It is intentionally small: one fight, one enemy, one deck, and a few combo patterns.

How to run:
1. Make sure Python 3 is installed.
2. From a terminal, run:

   python trial_roguelike_prototype.py

No extra packages are required. It uses tkinter, which ships with most Python installations.

What to test:

- Does Prior Statement -> Strike from Record feel satisfying after Confident Testimony?
- Is Objection fun and understandable?
